### 👋 About Me

Welcome to **Code Core Labs**! I specialize in bridging high-level **AI algorithms** and **full-stack web applications** with low-level **embedded hardware**. Whether it's deploying YOLO computer vision pipelines, building full-stack web platforms, or laying out custom PCBs, I enjoy crafting seamless end-to-end tech solutions.
